# Boston Housing - ITMO EPML

Проект курса **Engineering Practices for Machine Learning** (ITMO).

Задача - предсказание медианной стоимости жилья (в тыс. $) по датасету Boston Housing с применением лучших практик ML-инжиниринга.

---

## Что делает проект

- Обучает три модели scikit-learn (LinearRegression, Ridge, RandomForest)
- Управляет данными через **DVC** (версионирование, воспроизводимые пайплайны)
- Конфигурирует эксперименты через **Hydra** (composition, CLI-оверрайды)
- Трекает эксперименты в **MLflow** (локальный SQLite backend)
- Интегрируется с **ClearML** (enterprise MLOps: реестр моделей, пайплайны, агент)
- Публикует документацию через **MkDocs + GitHub Pages**

## Быстрый старт

```bash
git clone <repo-url>
cd ITMO-EPML-HW
poetry install
poetry run dvc pull
poetry run dvc repro
```

Смотрите [Руководство по развертыванию](deployment.md) для детальных инструкций.
